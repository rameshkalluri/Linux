# Day 3 — Users, Processes & Packages

Notes for Day 3 of the [5-Day Linux for DevOps Plan](../Linux-5-Day-Learning-Plan.md).

## Topics
1. [Users & Groups](01-users-and-groups.md) — accounts, `sudo`, `/etc/passwd`.
2. [Processes & Jobs](02-processes-and-jobs.md) — `ps`, `top`, `kill`, background jobs.
3. [System Monitoring](03-system-monitoring.md) — `free`, `df`, `du`, `uptime`, load.
4. [Package Management](04-package-management.md) — `apt`, `dnf`/`yum`, installing software.
5. [systemd & Services](05-systemd-and-services.md) — `systemctl`, `journalctl`.

## Day 3 Goals
- [ ] Create users/groups and grant `sudo` access.
- [ ] List, inspect, and kill processes; run background jobs.
- [ ] Check CPU, memory, and disk health (`free`, `df`, `top`).
- [ ] Install and remove software with the right package manager.
- [ ] Start/enable services and read their logs with `systemctl`/`journalctl`.
