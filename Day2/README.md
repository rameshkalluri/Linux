# Day 2 — Files, Text Processing & Permissions

Notes for Day 2 of the [5-Day Linux for DevOps Plan](../Linux-5-Day-Learning-Plan.md).

## Topics
1. [Viewing & Editing Files](01-viewing-and-editing.md) — `cat`, `less`, `head`, `tail`, `nano`, `vim`.
2. [Permissions & Ownership](02-permissions-and-ownership.md) — `rwx`, octal, `chmod`, `chown`, `umask`.
3. [Redirection & Pipes](03-redirection-and-pipes.md) — `|`, `>`, `>>`, `2>`, `/dev/null`.
4. [Text Processing](04-text-processing.md) — `grep`, `sed`, `awk`, `cut`, `sort`, `uniq`, `wc`.
5. [Finding Files](05-finding-files.md) — `find`, `locate`, `which`.

## Day 2 Goals
- [ ] Read and edit files in the terminal (incl. survive `vim`).
- [ ] Read `rwx`/octal permissions and fix them with `chmod`/`chown`.
- [ ] Redirect output to files and chain commands with pipes.
- [ ] Extract and transform text with `grep`, `awk`, `sed`, `cut`, `sort`, `uniq`.
- [ ] Locate any file on the system with `find`.
