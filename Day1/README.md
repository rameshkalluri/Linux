# Day 1 — Linux Fundamentals & The Filesystem

Notes for Day 1 of the [5-Day Linux for DevOps Plan](../Linux-5-Day-Learning-Plan.md).

## Topics
1. [What is Linux](01-what-is-linux.md) — kernel vs shell vs distro, why DevOps uses it.
2. [Filesystem Hierarchy](02-filesystem-hierarchy.md) — `/`, `/etc`, `/var`, FHS tour.
3. [Navigation & File Basics](03-navigation-and-files.md) — `pwd`, `ls`, `cd`, `cp`, `mv`, `rm`.
4. [Getting Help](04-getting-help.md) — `man`, `--help`, `tldr`, reading docs.

## Day 1 Goals
- [ ] Explain the difference between the kernel, shell, and a distro.
- [ ] Navigate the filesystem confidently using absolute and relative paths.
- [ ] Know what lives in `/etc`, `/var`, `/home`, `/bin`, `/proc`.
- [ ] Create, copy, move, and delete files and directories.
- [ ] Get unstuck on any command using `man` / `--help` / `tldr`.
